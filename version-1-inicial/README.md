# Geometric Figures – Initial Version

## Description

This project represents the first evaluable assignment developed at the beginning of the Programming module.

It is a console-based Java application that allows the user to calculate the area and perimeter of different geometric figures using an interactive menu.

The program interacts with the user through the console and validates input values to prevent incorrect calculations.

## Learning Objectives

At this stage of the course, the goal was to practice:

- Using Scanner for user input
- Control structures (switch, do-while, if)
- Variables and data types
- Basic mathematical calculations
- Input validation

Object-Oriented Programming concepts such as inheritance, abstract classes, and polymorphism had not yet been introduced.

## Technical Characteristics

- All logic is implemented inside the `main` method
- Each geometric figure is calculated directly within the `switch` statement
- No class separation or method abstraction
- Structured programming approach

This repository is intentionally kept in its original form to demonstrate the starting point of the learning process.

# Figuras Geométricas – Versión Inicial

## Descripción

Este proyecto representa la primera actividad evaluable desarrollada al inicio del módulo de Programación.

Se trata de una aplicación de consola en Java que permite calcular el área y el perímetro de diferentes figuras geométricas mediante un menú interactivo.

El programa interactúa con el usuario a través de la consola y valida los valores introducidos para evitar cálculos incorrectos.

## Objetivos de Aprendizaje

En esta etapa del curso, el objetivo era practicar:

- Uso de Scanner para entrada de datos
- Estructuras de control (switch, do-while, if)
- Variables y tipos de datos
- Cálculos matemáticos básicos
- Validación de datos

Aún no se habían introducido conceptos de Programación Orientada a Objetos como herencia, clases abstractas o polimorfismo.

## Características Técnicas

- Toda la lógica está implementada dentro del método `main`
- Cada figura geométrica se calcula directamente dentro del `switch`
- No existe separación en clases ni abstracción de métodos
- Enfoque de programación estructurada

Este repositorio se mantiene intencionalmente en su forma original para demostrar el punto de partida del proceso de aprendizaje.
